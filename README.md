### 周报写作流程

（1）在【Issues】建立“issue+日期”命名,日期为当周周五的日期，比如“issue181019”,将每日有触动的信息放在isuue里。
（2）到周五时，在【docs/】里建立“issues181019.md”，整理【Issues】里【issue181019】的信息为文章。
（3）发表在自己的其他账号里，比如简书，知乎等等。
